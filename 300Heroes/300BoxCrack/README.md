 # 300Heroes Box Moekai Launcher
 #### version 1.2.0.2
 ----------

### 使用方式

 * 将 300BoxMoekaiLauncher 放至300Box盒子安装目录内
 * 直接运行即可



### 其它说明

 * 同目录下 300boxmkp.ini 文件为必要的配置文件，如果看得懂可以自己修改
 * 更新配置后删除boxmd5配置组的内容进行重新初始化
